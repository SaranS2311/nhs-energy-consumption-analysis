## 🧾 Project Overview

This group project focuses on analysing **energy consumption data within NHS facilities in Swansea** to identify cost-saving opportunities and improve operational efficiency.

Hospitals consume large amounts of energy, leading to high operational costs and significant environmental impact. This project transforms raw energy data into meaningful insights to support better decision-making — helping the NHS reduce costs while working towards a greener future.

## 🧠 Methodology: Six Sigma — DMAIC

| Phase | What We Did |
|-------|-------------|
| 📌 **Define** | Identified energy-related problems in NHS facilities |
| 📏 **Measure** | Collected and prepared raw energy consumption data |
| 🔍 **Analyze** | Identified trends, patterns and inefficiencies |
| 💡 **Improve** | Suggested targeted optimisation strategies |
| ✅ **Control** | Outlined measures to ensure long-term improvements |
